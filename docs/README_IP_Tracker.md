## IP Tracker

A clean and powerful Python tool that tracks IP addresses and visualizes their geographic location on an interactive map.

Ideal for OSINT exercises, networking projects, or simple curiosity-driven exploration.

---

## Notice ⚠️ 

This project uses the public API from ip-api.com under their free usage policy (maximum 45 requests per minute).  
For higher limits or commercial usage, please refer to their premium plans.

The tool is built using Folium, ip-api.com, and a touch of terminal styling via pyfiglet.

---

## Features

- Clean terminal interface with ASCII banner
- Retrieves detailed IP information (city, region, country, ISP, etc.)
- Generates interactive maps with a marker at the IP location
- Saves each lookup as an `.html` file viewable in a web browser
- CLI-compatible – usable via terminal input or arguments
